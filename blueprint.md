# Fortune Cookie App

## Overview

This is a simple web application that displays a random fortune cookie message when a button is clicked.

## Project Structure

*   `index.html`: The main HTML file.
*   `style.css`: The CSS file for styling.
*   `main.js`: The JavaScript file for functionality.

## Features

*   Displays a random fortune from a predefined list.
*   A button to generate a new fortune.

## Current Plan

*   **Objective:** Create a fortune cookie website.
*   **Steps:**
    1.  Modify `index.html` to create the basic structure of the fortune cookie page.
    2.  Style the page using `style.css`.
    3.  Implement the fortune generation logic in `main.js`.
